Temp Mail API Docs

A powerful and simple REST API for temporary email service.

https://api.barid.site

GET /domains

Retrieve a list of supported email domains.

curl https://api.barid.site/domains

GET /emails/:emailAddress

Retrieve a paginated list of email summaries for a specific recipient email address.

:emailAddress path required The full email address to query
limit query Maximum number of emails to return (default: 10)
offset query Number of emails to skip for pagination (default: 0)
curl "https://api.barid.site/emails/user@example.com?limit=5&offset=0"

DELETE /emails/:emailAddress

Delete all emails associated with a specific recipient email address.

:emailAddress path required The full email address whose emails are to be deleted
curl -X DELETE https://api.barid.site/emails/user@example.com

GET /inbox/:emailId

Retrieve the full content of a specific email by its unique ID.

:emailId path required The unique identifier of the email
curl https://api.barid.site/inbox/some-email-id

DELETE /inbox/:emailId

Delete a specific email by its unique ID.

:emailId path required The unique identifier of the email to be deleted
curl -X DELETE https://api.barid.site/inbox/some-email-id
